VISEPOCH documentation
======================

`VISEPOCH <https://github.com/georgeholt1/visepoch/>`_ is a Python 3 package for
analysing, interpreting and visualising
`EPOCH <https://github.com/Warwick-Plasma/epoch>`_ particle-in-cell laser-
wakefield acceleration (LWFA) simulation data.
